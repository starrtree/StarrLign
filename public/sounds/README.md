# StarrLign Sound Files

Add your custom sound effects in this folder with these exact filenames:

- `task-complete.mp3` — played when a task is completed.
- `subtask-toggle.mp3` — played when a subtask is checked/unchecked.
- `subtask-complete.mp3` — bonus pop/chime for checking a subtask complete.
- `project-complete.mp3` — bigger celebration when all active tasks in a project are done.

Recommended format:
- File type: MP3
- Sample rate: 44.1kHz
- Bitrate: 128-192 kbps CBR
- Length:
  - `task-complete.mp3`: 350ms to 1200ms
  - `subtask-toggle.mp3`: 80ms to 400ms
  - `subtask-complete.mp3`: 120ms to 500ms
  - `project-complete.mp3`: 600ms to 1800ms

Keep clips short and punchy to avoid UI lag.
